# Projeto de Validação de Formulário
Este é um projeto de validação de formulário desenvolvido em JavaScript, CSS e HTML. 
O objetivo é fornecer uma funcionalidade de validação de campos em um formulário web, garantindo que as informações sejam preenchidas corretamente antes de serem enviadas.

# Tecnologias Utilizadas
JavaScript: utilizado para implementar a lógica de validação e interatividade do formulário. 
CSS: utilizado para estilizar o formulário e fornecer uma experiência visual agradável. 
HTML: utilizado para criar a estrutura do formulário e seus elementos.

# Funcionalidades
Validação de campos obrigatórios: garante que os campos obrigatórios sejam preenchidos antes de enviar o formulário. 
Validação de formatos: verifica se os campos estão preenchidos com os formatos corretos, como endereço de e-mail ou número de telefone. 
Feedback visual: fornece feedback visual instantâneo aos usuários, destacando campos inválidos e exibindo mensagens de erro.

# Como Usar
Faça o download ou clone o repositório do projeto. 
Abra o arquivo HTML em seu navegador. Preencha os campos do formulário. 
Ao tentar enviar o formulário, as validações serão acionadas e mensagens de erro serão exibidas, caso necessário.
